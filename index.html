Why VigiSense?

8 billion pounds is the cost of injuries due to work place accidents in the UK. Clearly there needs to be a solution.

What does Vigisense offer?

Real time monitoring of biological signals that are telltale signs of accidents. This system will detect the incidence of accident causing conditions rather the incidence of an accident itself. Whats on offer:

  • Low processing needed
  • Low energy consumption
  • Network integration
  • Minimal Hardware
  • Portability
  • Situation Classification

With our modular design, the whole thing can be percisely customized to suit any application.

How does it work?

Interupt based data acquisition, enabling responsive user interface.

Sensor Selection

For each application, critical biosignals are measured using carefully chosen sensors.

Real time processing

Efficient algorithms enable swift processing of data at limited processing capacity and memory needs.

Situation appropriate response

The system can be integrated to suit the user framework and respond according to set protocols of safety.
